### 初始化React项目

1- 建议使用 [Create React App](https://create-react-app.dev/) 初始化项目

- typescripts + npm

```
npx create-react-app demo --template typescript --use-npm
cd demo
npm start
```

- 访问项目 http://localhost:3000/

### Optional

- 添加webpack [Create App](https://createapp.dev/)
